### Conteúdo do Curso (Seção 5)
**Objetivo**: Mostrar o valor completo

#### Headline
```
O Que Você Vai Aprender
```

#### Módulos (Accordion ou Tabs)

**Módulo 1: Setup & Ferramentas** (1-2h)
- Configuração completa de ambiente
- Docker, Git, VS Code, Java SDK Manager
- Primeiro projeto no ar

**Módulo 2: Git Profissional** (3-4h)
- Conventional Commits
- Copilot para commits automáticos
- Rebase e histórico organizado
- Pull Requests profissionais

**Módulo 3: Design com IA** (4-5h)
- Stitch: gerar design em minutos
- Figma básico para ajustes
- Do wireframe ao protótipo finalizado
- Sem ser designer

**Módulo 4-5: Frontend** (14-18h)
- HTML, CSS, JavaScript (fundamentos)
- Next.js (React explicado dentro)
- Arquitetura HFSA
- Deploy na Vercel

**Módulo 6: IA no Desenvolvimento** (5-6h)
- Workflow completo com IA
- Documentação automática
- Kanban em Markdown
- User Stories com IA
- TDD com IA

**Módulo 7-8: Backend** (14-18h)
- Java puro (antes do framework!)
- Spring Boot
- API REST profissional
- Autenticação e segurança (JWT)

**Módulo 9-10: Infraestrutura** (10-13h)
- Docker e Docker Compose
- Banco de dados (PostgreSQL)
- Nginx como proxy reverso
- Docker Swarm (escalabilidade)

**Módulo 11: Documentação & Testes** (5-6h)
- Swagger com IA
- TDD (testes antes do código)
- Integração de testes no workflow

**Módulo 12: Projeto Final** (4-6h)
- Aplicação completa: Frontend + Backend
- Sistema de gerenciamento com autenticação
- Deploy profissional

**Total**: 66-88 horas de conteúdo

#### Visual
- Ícones para cada módulo
- Progress bar mostrando a jornada

## Tecnologias Sugeridas

### Opção 1: Next.js (Código Próprio)
**Prós**: Controle total, SEO excelente, performance
**Contras**: Mais tempo de desenvolvimento
**Recomendado se**: Thiago tem tempo e quer mostrar habilidade

### Opção 2: Framer
**Prós**: Rápido, bonito, sem código, animações
**Contras**: Custo mensal, menos customização
**Recomendado se**: Priorizar velocidade

### Opção 3: Webflow
**Prós**: Visual, CMS integrado, responsivo fácil
**Contras**: Curva de aprendizado, custo
**Recomendado se**: Quer CMS para blog futuro

### Opção 4: WordPress + Elementor
**Prós**: Familiar, plugins, barato
**Contras**: Pode ficar pesado, manutenção
**Recomendado se**: Já conhece bem WordPress

## Testes A/B Sugeridos

### Headlines
- Testar 3 variações diferentes
- Medir taxa de permanência na página

### CTAs
- Cor do botão (verde, azul, laranja)
- Texto ("Matricular Agora" vs "Começar Agora" vs "Quero Aprender")

### Prova Social
- Com/sem número de alunos
- Com/sem rating
- Com/sem logos de empresas

### Garantia
- 7 dias vs 14 dias vs 30 dias (se possível)

### Preço
- Mostrar preço "De-Por" vs apenas preço final
- Destacar parcelamento vs preço total
